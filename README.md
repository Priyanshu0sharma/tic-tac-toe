# Tic Tac Toe

3D Tic Tac Toe game with Local, Robot (Easy/Hard), and Online modes.

## Features
- Dark/Light theme toggle
- Robot mode with Easy (random) and Hard (Minimax AI)
- Highlight winning cells
- Local 2-player mode
- Online mode with Create/Join game using Firebase
- Back buttons on all pages
- 3D animated boxes and hover effects
